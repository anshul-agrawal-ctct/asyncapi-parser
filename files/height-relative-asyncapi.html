Height Relative API 0.0.1

This API is used to define messages related to height relative information. Height relative information is used to locate a part of a machine in terms of height compared to a reference. For example, a laser or sonic device that provides a height from a reference offers height relative capabilities.

Servers

  • MQTT://server.url/ MQTT 5.0 Open Grade Control API Server

Operations

onActivateSource

Receive activeSource

Description:
Channel used to activate a specific height relative source by its id (uuid). If a HR source matches the id, it is expected to report as active and start sending pose data. All other available HR sources must stop sending pose data and report as inactive.

Accepts the following message:
  • SchemaactivateSource
    Schema Formatfbs
    ActivateHeightRelativeSource
    Field Name Type
    uuid string
onVerticalOffsetReceived

Receive verticalOffsets

Description:
Channel used to tell the HR source to offset the height measurements.

Accepts the following message:
  • SchemaverticalOffsets
    Schema Formatfbs
    HeightRelativeVerticalOffsets
    Field Name Type
    uuid string
    left_blade_tip_height_offset double
    right_blade_tip_height_offset double
onBenchRequestReceived

Receive benchRequest

Description:
Channel used to tell a specific HR to bench.

Accepts the following message:
  • SchemaheightRelativeVerticalOffsets
    Schema Formatfbs
    HeightRelativeBenchRequest
    Field Name Type
    uuid string
pose

Send pose

Description:
Channel used for pose data from the active height relative source.

Accepts the following message:
  • Schemapose
    Schema Formatfbs
    HeightRelativePose
    Field Name Type
    uuid string
    pose_time_stamp Core
    left_blade_tip_height double
    right_blade_tip_height double
report

Send report

Description:
Channel used for height relative sources to post their report.

Accepts the following message:
  • Schemareport
    Schema Formatfbs
    HeightRelativeReport
    Field Name Type
    uuid string
    name string
    manufacturer_name string
    manufacturer_code uint
    source_type HeightRelativeSourceType
    svg_icon string
    operator_ui_base_url string
    technician_ui_base_url string
    status Status
