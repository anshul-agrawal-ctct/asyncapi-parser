Site Relative API 0.0.1

This API is used to define messages related to site relative information. Site relative information is used to locate a part of a machine (platform or attachment) on the site, using local site coordinates. For example, a software component that consumes GNSS data (LLH) and convert it to output NEE data offers site relative capabilities.

Servers

  • MQTT://server.url/ MQTT 5.0 Open Grade Control API Server

Operations

onActivateSource

Receive activeSource

Description:
Channel used to activate a specific site relative source by its id (uuid). If a SR source matches the id, it is expected to report as active and start sending pose data. All other available SR sources must stop sending pose data and report as inactive.

Accepts the following message:
  • SchemaactivateSource
    Schema Formatfbs
    ActivateSiteRelativeSource
    Field Name Type
    uuid string
onActivateCalibration

Receive activeCalibration

Description:
Channel used to activate a new local site calibration file. Payload includes paths and hashes for download through the O&S REST api.

Accepts the following message:
  • SchemaactivateCalibration
    Schema Formatfbs
    ActivateSiteRelativeCalibration
    Field Name Type
    uuid string
    localization_file_url string
    localization_file_hash string
    grid_file_url string
    grid_file_hash string
pose

Send pose

Description:
Channel used for pose data from the active site relative source.

Accepts the following message:
  • Schemapose
    Schema Formatfbs
    SiteRelativePose
    Field Name Type
    uuid string
    pose_time_stamp Core
    reference PoseReference
    nee SiteRelative
    llh Core
    orientation Core
    travel_direction Core
    pitch Core
    roll Core
report

Send report

Description:
Channel used for site relative sources to post their report.

Accepts the following message:
  • Schemareport
    Schema Formatfbs
    SiteRelativeReport
    Field Name Type
    uuid string
    name string
    manufacturer_name string
    manufacturer_code uint
    source_type SiteRelativeSourceType
    svg_icon string
    operator_ui_base_url string
    technician_ui_base_url string
    status Status
