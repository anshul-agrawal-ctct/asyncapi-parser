Machine Positioning Source 0.0.1

This API is used for machine positioning information related messages.

Servers

  • MQTT://server.url/ MQTT 5.0 RA-API

Operations

    Schema Formatfbs

    Field Name Type

pose

Send pose

Description:
This channel is used to publish positions of the various parts of the machine for guidance and rendering. Published at 10Hz. Contains the merged pose of rigid body elements with site relative pose, along with extra fixed points from the measure up, like left and right blade tips. Has a machine type generic section for platform and attachment positions and blade tips and a machine specific section, with positions for extra ridged bodies, like boom, c-frame etc. Also contains a position ID (or epoch) for the MPS and SR positions and measureup ID used to calculate each positon message

Accepts the following message:
  • Schemapose
    Schema Formatfbs
    DozerMachinePose
    Field Name Type
    c_frame Core
    push_arm Core
    ExcavatorMachinePose
    Field Name Type
    boom Core
    va_boom Core
    stick Core
    coupler Core
    slew_angle Core
    MachinePose
    Field Name Type
    pose_id uint8
    pose_time_stamp Core
    left_blade_tip Core
    right_blade_tip Core
    attachment Core
    platform Core
    llh Core
    machine_specific_pose MachineSpecificPose
